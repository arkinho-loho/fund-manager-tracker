# Changelog

## v1.2.0 (2026-06-27)

### Added
- 首次使用引导模式（`references/onboarding-guide.md`）
- 整体股票仓位变化追踪（Q→Q）
- 风格分组展示（AI产业链/科技成长/消费价值/深度价值/周期/QDII等 7 组）
- 业绩基准标注到每位经理
- 关注理由/评价列添加到总览表

### Changed
- 标杆基金数量从每组 1 只扩展到 1-3 只
- 观点从简短摘要扩展为完整段落引用，按核心主题分组
- 输出格式从 MD+HTML 双版本改为仅 MD
- 所有硬编码路径替换为环境变量占位符
- `manager-watchlist.json` 新增 `style_group`、`benchmark`、`watch_reason` 字段

### Fixed
- 已离职经理生命周期管理（查新公募 → 查私募 → 继续观察）
- 跨期持仓对比缺失 → 接入天天基金网 `FundArchivesDatas.aspx` API
- 数据质量报告从文件输出改为仅对话输出
